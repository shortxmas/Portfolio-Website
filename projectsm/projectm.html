LockBox

Java

May, 2023

LockBox is a small and compact program that allows the user to password protect and lock any folder of their wish. The program works by changing Windows folder attributes and creating a hidden key within the folder which will later be used to unlock that folder. Simply drop the LockBox file into the folder you want to lock and follow the directions in the program.
